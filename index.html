Academic Screen Share

An Introduction to Screen Sharing

The intention of the academic screen share project is to help you select web based resources that can support small group tasks within a large class setting. For example, it is useful to work on an enquiry based task in a small group. e.g. the design or plan for a solution to a case study problem and present the groupâ€™s response to the rest of the class. Software must have certain key features in order to support small group learning tasks within a large class setting such as enabling students to:

  • Communicate ideas
  • Create a file such as an image or a document
  • Review/edit the file or comment on it
  • Make available a finished file or view that can be projected on to the screen in a large class

The software should also be easy to use and accessible by mobile devices.

The links on the page will show you quick start guides to various web and cloud apps.
